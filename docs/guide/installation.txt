Installation
============

Requirements
------------

Python 3.10 to 3.13 supported.

Django 4.2 to 5.2 supported.

Installation
------------

1. Install with **pip**:

   .. code-block:: sh

       python -m pip install django-resume

2. Add django-resume to your ``INSTALLED_APPS``:

   .. code-block:: python

       INSTALLED_APPS = [
           ...,
           "django_resume",
           ...,
       ]

django-resume's editable templates load HTMX where needed. You do not need to
install or configure django-htmx to use the built-in editing views.
